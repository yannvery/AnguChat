## AngularJS - Rails - Bootstrap

Simple Chat with AngularJS and testing

[AngularJS](http://angularjs.org/)

## HowTo

	git clone git@github.com:yannvery/AnguChat.git anguchat
	cd anguchat
	
	# ruby 2.0.0 needed
	
	gem install bundler
	
	bundle install
	rake db:migrate
	bundle exec rails s

Open web browser @ http://localhost:3000/
